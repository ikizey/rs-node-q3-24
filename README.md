# Battleship Game task

## Installation

1. Clone/download repo
2. `npm install`

## Usage

**Development**

`npm run start:dev`

- App served @ `http://localhost:8181` with nodemon

**Production**
`npm run build`
`npm run start:prod`

- App served @ `http://localhost:8181` as a production build
